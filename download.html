Downloads

This page gives information on how to download Vampire. See Usage for how to use Vampire.

Binaries

The binaries of the new Vampire can be downloaded and used under the terms of the Vampire License:

  • Linux 64-bit binaries
    • Version 4.1
    • Version 4.1 (without Z3)
    • CASC Version 4.0
    • SMT-COMP Version X
  • MacOS binaries
    • Version 4.1
    • Version 4.1 (without Z3)
    • CASC Version 4.0
    • SMT-COMP Version X
All binaries are statically linked. However, on Mac there are some issues with this and you may need to compile directly from source to link against the appropriate libraries.

Source

The source of Vampire can be obtained from GitHub either by cloning the repository or as a Zip download.
